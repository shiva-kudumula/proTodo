# ProTodo

A responsive Todo List web application built using HTML, CSS, JavaScript, and Bootstrap.

## Live Demo

https://shiva-kudumula.github.io/proTodo/

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Save tasks using Local Storage
* Responsive user interface
* Tasks persist after page refresh

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 4
